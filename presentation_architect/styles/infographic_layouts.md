# Infographic Structural Layouts

This file defines 20 structural layouts for presentation infographics. Use this reference in Phase 2 to determine the correct `Assigned Layout` based on the semantic meaning and logical structure of the content.

| Layout | Semantic Trigger (Best For) |
|--------|----------|
| `bridge` | Problem-solution, gap-crossing, transitioning from current state to future state |
| `circular-flow` | Cycles, recurring processes, continuous feedback loops |
| `comparison-table` | Multi-factor comparisons across different entities or products |
| `do-dont` | Correct vs incorrect practices, best practices guidelines |
| `equation` | Formula breakdown, input-to-output transformations |
| `feature-list` | Product features, simple bullet points, unranked lists |
| `fishbone` | Root cause analysis, cause-and-effect breakdowns |
| `funnel` | Conversion processes, filtering stages, sales pipelines |
| `grid-cards` | Multiple independent topics, high-level overviews, Bento box UI |
| `iceberg` | Surface vs hidden aspects, what is visible vs underlying infrastructure |
| `journey-path` | Customer journey, winding roadmaps, sequential milestones |
| `layers-stack` | Technology stack, architectural layers, foundational dependencies |
| `mind-map` | Brainstorming, idea mapping, radial associations |
| `nested-circles` | Levels of influence, scope, core vs periphery |
| `priority-quadrants` | Eisenhower matrix, 2x2 grids, mapping on two axes |
| `pyramid` | Hierarchy, Maslow's needs, bottom-up foundational structures |
| `scale-balance` | Pros vs cons, weighing two opposing options |
| `timeline-horizontal` | History, strictly chronological events, linear roadmaps |
| `tree-hierarchy` | Org charts, taxonomies, top-down categorization |
| `venn` | Overlapping concepts, intersecting domains |
